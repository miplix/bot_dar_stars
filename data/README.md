# Директория для базы данных

В этой директории будет храниться SQLite база данных бота.

## Файлы

- `bot_database.db` - основная база данных (создается автоматически при первом запуске)

## Структура таблиц

### users
Информация о пользователях бота

### calculations
История расчетов даров

### gifts_knowledge
База знаний о дарах (1-22)

### ai_interactions
История взаимодействий с ИИ

### payments
История платежей и подписок

## Резервное копирование

Рекомендуется регулярно создавать резервные копии базы данных:

```bash
cp bot_database.db bot_database_backup_$(date +%Y%m%d).db
```
